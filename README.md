# smart-reminder-extension
A Chrome extension for scheduled and recurring reminders with missed alert handling.

---

## ✨ Features

- Set reminders with date & time  
- Repeat reminders (daily / weekly)  
- Automatically shows missed reminders when Chrome reopens  
- Smart retry (reminds again every 30 minutes if ignored)  
- "Mark as Done" button  
- View active and past reminders  
- Modify or delete reminders  
- Clean and simple UI

## 🚀 Installation

1. Download or clone this repository  
2. Open Chrome and go to:  
   chrome://extensions/  
3. Enable Developer Mode  
4. Click "Load unpacked"  
5. Select this folder
